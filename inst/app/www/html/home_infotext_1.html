In the options to the left of this info box, you see a dropdown after succesfully connecting. Here you first need to select and load the project database.

After the index of your project has loaded, you are prompted to select a Place from the next dropdown.
The list will show all Place-resources in the project database that contain so called Operations. In the case of Operations that are not located inside a Place such as the Steindepot they will be shown here.

After selecting the Place, you can select an Operation. Operations are Trenches or Surveys.
